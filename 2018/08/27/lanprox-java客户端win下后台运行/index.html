lanprox java客户端win下后台运行

##lanproxy是什么?
lanproxy 与frp和socks或者N2n的反向代理的一种工具

##lanproxy有哪些优点？

  • 方便添加映射端口
    • 不像frp，需要在客户端上手动修改配置才能完成映射。
    • lanproxy，可以直接在java环境下运行
  • lanproxy 支持ssl加密传输。
  • 界面美观

#Lanproxy下后台运行
